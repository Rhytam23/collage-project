# 🚀 Level Up AI: Learning Nexus 2.0

> [!TIP]
> **Project Goal**: To bridge the gap between static learning and personal growth through an AI-driven, adaptive difficulty system.

Developed for the **HACK_O_NIT 2026** Hackathon, **Level Up AI** is a premium, gamified learning platform designed to evolve with the user. The platform tracks your cognitive peaks and logical friction points to dynamically scale question difficulty, ensuring you stay in the "Flow State."

---

## ✨ Key Features

### 🧠 Dynamic AI Difficulty Manager
Our core engine monitors your correct/incorrect streaks in real-time. If you're crushing it, the AI scales up to **Hard Mode**; if you're struggling, it recalibrates to reinforce your core foundations.

### 🎮 Gamified Ecosystem
- **XP Progression**: Each correct answer fuels your ascent through the "Neural Levels."
- **Daily Streaks**: Encourages consistent learning habits with visual feedback.
- **Global Leaderboard**: Compare your standings against other "Architects" and "Data Weavers."

### 💎 High-Fidelity UI/UX
Designed with a "Hackathon Professional" aesthetic:
- **Bento-Grid Architecture**: Precise information density and clean layouts.
- **Glassmorphism**: Sophisticated blur and border effects.
- **Framer Motion**: Smooth entrance and transition animations.

---

## 🛠️ Tech Stack

- **Framework**: React 19 + Vite 8
- **Styling**: Tailwind CSS v4 (Alpha/PostCSS)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Avatars**: Dicebear API

---

## 🚦 Getting Started

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Rhytam23/hack_o_nit_2026.git
    cd level-up-ai
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Launch the Arena**:
    ```bash
    npm run dev
    ```
